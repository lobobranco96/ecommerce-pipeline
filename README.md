# ecommerce-pipeline

# Stack:
  - docker-compose
  - airflow 3.0
  - minio-buckets: raw, processed
  - great_expectation - data quality
  - metabase
  - python 3.12
  - spark - distributed processing
  - file_format: csv, parquet
  - datawarehousing: postgres
  - grafana + prometheus: metrics from airflow, spark, docker visualization
